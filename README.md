## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd elegance-fashion
   ```

2. **Install dependencies**
   ```bash

   ```

   ```bash
   npm run dev
   ```
